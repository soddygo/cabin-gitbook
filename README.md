#基于Cabin,自己封装或开发的插件以及模板说明

##介绍
本文档使用gitbook编写,下载此项目,项目根目录,需要用node安装gitbook命令,然后运行命令:gitbook server,然后根据提示打开浏览器查看文档.


##插件项目地址
最新封装项目代码,github地址:

    https://github.com/soddygo/date-pick
